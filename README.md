# redmine_jp_address
Redmine用の日本向けのアドレス帳です.

## Features
- プロジェクトと無関係のグローバルなアドレス帳
- 名前とかなのほか、複数の住所・電話番号・メールアドレスを登録できる
- 住所情報は日本向け
- 法人のアドレスに対応
- 名前・かな・電話番号・メールアドレスで絞り込み検索
- Wiki的なメモ
- プロジェクトごとのアドレス帳
- 親プロジェクトのアドレスも表示
- グローバルなアドレス帳からプロジェクトのアドレス帳に簡単に選択して追加
- プロジェクトのオーナーをプロジェクト名に表示する(選択可)

## Requirement
- Redmine 3.0.1
